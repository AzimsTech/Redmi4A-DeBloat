# [Magisk] DeBloater for Redmi 4A (MIUI 9 & 10)

_Hides unnecessary apps from MIUI._

![](cover.jpg)
## Description
This Magisk module replaces/removes MIUI bloatware with an empty directory so the system won't be able to see it.
